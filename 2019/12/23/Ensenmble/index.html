Ensenmble learning

Posted on 2019-12-23 Edited on 2019-12-27

In statistics and machine learning, Ensemble method use multiple learning algorithms to obtain better predictive performance than could be obtained from any of the constituent learning algorithms alone.

For regression task, given a set of regressors $r_i \in \mathcal{R}$ and training data $T$, every $r_i$ can be trained on $T$ and results in a single regression function

, denoted as $g_i(X)$. Similarly for classification, given a set of classifiers $c_i \in \mathcal{C}$, every $c_i$ can be trained on $T$ and results in a classification function that estimate the probability(or equivalent) of a given input belonging to every class

, $\mathcal{P}(X|c_i,T)$ denoted as $g_i(X)$ which returns a vector of probs. Hense, classification task can also be taken as multi-variate linear regression where the dependent variable is the conditional probability $P(X|y,c_i,T)$.

Either way, Ensemble learning integrated these learning algorithms by taking a weighted average of $g_i(X)$.

Hard & soft voting

Voting is an idea specific for classification. every learning algorithms is trained on the whole dataset and averaged with equal weight.

The soft voting is directly taking the average of prob vectors while the hard voting instead takes the average of one-hot vector transformed from probs vectors. For example, in one binary classification task we ensemble three classifiers, the prob that an observation belongs to class 1 equal (0.45, 0.45, 0.9). With soft voting the average prob is (0.45 + 0.45 + 0.9)/3 = 0.6 while hard voting will return the average prob equal (0 + 0 + 1)/3 = 0.33.

Bagging

Bootstrap aggregating, often abbreviated as bagging, involves having each model in the ensemble vote with equal weight. In order to promote model variance, bagging trains each model in the ensemble using a randomly drawn subset of the training set.

Boosting

Boosting involves incrementally building an ensemble by training each new model instance to emphasize the training instances mis-classified by previous models. In some cases, boosting has been shown to yield better accuracy than bagging, but it also tends to be more likely to over-fit the training data.
